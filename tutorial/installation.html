Installation

Setting up WEBMOL is straightforward. Since it uses native PHP Core, there are no heavy environment requirements. You just need PHP 8.1 or higher.

1

Clone the Repository

$ git clone https://github.com/your-username/webmol.git
$ cd webmol
$ php -S localhost:8000 -t public
2

Database Setup

Open your config/database.php file to define your molecular connection.

PHP
// config/database.php
return [
    'driver'   => 'mysql',
    'host'     => '127.0.0.1',
    'database' => 'webmol_app',
    'user'     => 'root',
    'pass'     => 'secret'
];
