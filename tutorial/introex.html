Introduction

Welcome to the WEBMOL tutorial. This guide will walk you through building your first molecular application using PHP Core.

First Request Handling

Defining a simple route within the WEBMOL kernel.

PHP
<?php

use Webmol\Core\Router;

// Initialize the Route Molecule
$app = new Router();

$app->get('/hello', function() {
    return "Welcome to the Molecular Web.";
});

$app->dispatch();
